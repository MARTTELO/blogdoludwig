Estrutural Condicional


Para saber se um número é par ou ímpar deve-se fazer a divisão inteira do número por 2.

Se o resto da divisão é 0 (zero) o número é par, senão o número é ímpar.



//Exemplo while em C
  
    #include ‹stdio.h›

    void par_impar(){
    
    int n;
    
    printf("Aula 10 - Exemplo com If e Else \n \n");
    
    printf("Digite um número: ");
    scanf("%d", &n);
    
    if (n % 2 == 0){
        
        printf("%d é par \n",n);
        
    }else{
        
        printf("%d é impar \n",n);
    }
}

Estrutura de Repetição

While

A técnica denominada estrutura de repetição ou laço de repetição, é uma estrutura de programação que facilita repetir determinados trechos de código. Essa técnica reduz o trabalho de programação, principalmente quando é preciso repetir várias vezes alguma ação importante no programa.

A estrutura de repetição faz a verificação da condição para executar as instruções subordinadas. Caso o resultado lógico seja verdadeiro ele executa a ação do próximo bloco de instruções, senão ele sai do programa.



//Exemplo while em C
  
    #include ‹stdio.h›

    void aula12_2_tabuada_1_10_while() {


    int cont1, cont2;

    printf("**** Aula 12: Tabuada com While ****\n\n");

    cont1 = 1;
    cont2 = 0;

    while (cont1 <= 5) {
        printf("-----------------------------\n");
        printf("TABUADA DO NÚMERO: %d \n", cont1);
        printf("-----------------------------\n");

        while (cont2 <= 10) {
            printf("%d  x  %d  =  %d \n", cont1, cont2,\n
            cont1 * cont2);
            cont2 += 1;
        }
        cont2 = 0;
        cont1 += 1;

    }
    printf(" \n");
    printf(" Fim da Tabuada! \n");

}


//Fim do código

Estrutural de Repetição FOR


O laço com FOR é geralmente usado quando você sabe o número de iterações de antemão. Diferentemente do WHILE onde não sabemos esse número.

A estrutura de repetição pode ser classificada como Interativa e Iterativa. Interativa quando necessitam necessitam da intervenção de um usuário do programa, para repeti a próxima ação de um indeterminado número de vezes, Iterativa quando executam as repetições previstas de forma automática, conforme configuração da condição.
