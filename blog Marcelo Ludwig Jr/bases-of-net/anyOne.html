O que é AnyOne?

AnyOne é um sistema web para entrega de mercadorias com a possibilidade de escolher o dia e a hora da entrega. O usuário acessa o site preenche os dados de origem, destino, data, hora e clicando no botão calcular o frete, uma API do google entra em ação e traça a rota e oferece a distância. Com base nessa informação o sistema gera o valor do frete. O entregador recebe a mesagem via aplicativo e pode aceitar ou não aquele pedido. O sistema possui dois ambientes, um para o usuário e outro para o entregador, que pode fazer um relatório das entregas feitas.
