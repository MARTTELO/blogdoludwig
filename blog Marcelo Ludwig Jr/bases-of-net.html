Muitos aspectos importantes ocorreram desde então e os mais importantes são destacados no curso. O que é a internet?

A Internet é a maior rede de comunicações do mundo. Até (2014), a internet possuia mais de 2,4 bilhões de usuários conectados através de computadores, smartphones, tablets, celulares e outros dispositivos. Ela também é conhecida como “rede das redes”, pois é um conglomerado de redes menores que se interligam através de diversos meios e formam a Internet, uma rede em escala global. A evolução da internet tem sido exponencial e é praticamente impossível viver sem ela, está presente em quase tudo a nossa volta.

A internet é formada por:

  • Peopleware: Pessoas que fazem uso da internet.
  • Hardware: Dispositivos físicos que forma a rede: o computadores, dispositivos móveis, cabos, antenas, etc.
  • Software: Programas que usam a rede para a troca de informações.
  • Protocolos: Um conjunto de regras ou acordos para que dois computadores distintos se comuniquem. A Internet só é viável devido aos muitos protocolos que fazem a comunicação entre diferentes redes.

Tipos de Protocolos

Para que haja uma interação entre dois sistemas diferentes precisamos de um "intérprete". Cada protocolo trata de um elemento especifico nessa comunicação. A seguir uma pequena lista dos protocolos mais usuáis no nosso dia a dia:

  • TCP:Transmission Control Protocol

    > Protocolo que controla a transmissão de dados entre hosts (fim a fim). Uma de suas característica mais marcantes é que há uma garantia de entrega de pacotes.

  • • UDP: User Datagram Protocol

    > Tem a mesma função do TCP, porém, não há garantia de entrega de pacotes.

  • • IP: Internet Protocol

    > Basicamente, o protocolo IP provê endereçamento e encaminhamento de pacotes em uma rede de computadores. Cada host de uma rede possui um endereço IP.

  • • DNS: Domain Name System

    > É um protocolo hierárquico que armazena e divulga a relação entre endereços IP e domínios. Toda vez que um site é acessado, o DNS entra em ação, ou seja, nome do site é convertido para um número IP e assim é feita a requisição do site ao servidor.

  • HTTP: Hyper Text Transfer Protocolo

    > É um protocolo que possibilita a transferência de dados hipermídia – som, texto, imagem e vídeo. É usado na Web pelos browsers, por exemplo, ao acessar um site ou serviço na Web.

  • FTP: File Transfer Protocol

    > Usado para a transferência rápida de dados. Usado para fazer downloads de arquivos com serviços como o Dropbox, Skydrive e Google Drive

  • SMTP: Simple Mail Transfer Protocol

    > Protocolo padrão para o envio de e-mails através da Internet. Usado em aplicações de e-mail como Outlook, Thunderbird, entre outros.

  • POP3: Post Office Protocol

    > Protocolo padrão para acesso remoto à uma caixa de e-mail, ou seja, permite a transferência de arquivos de e-mail contidos em um servidor. Usado por aplicações como Outlook, Thunderbird, entre outros.
